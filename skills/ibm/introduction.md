`/ibm` is a quick-reference reminder for the Issue-Branch-Merge workflow -- it verifies you have an issue, a properly named branch, and a linked PR/MR before any code is written. Use it when you are about to start work and want a checklist to make sure nothing is missed. It is a cheat sheet, not an executor -- it tells you what to do, not does it for you. For the full workflow walkthrough, run `/ccwork tour workflow`.
