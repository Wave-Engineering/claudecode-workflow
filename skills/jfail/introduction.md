`/jfail` fetches and analyzes a failed CI job (GitLab) or GitHub Actions workflow run, summarizing the failure without flooding your context window. Use it when a pipeline fails and you want a quick diagnosis -- pass the job ID or URL and it will pull the logs, identify the root cause, and suggest a fix. It works with both GitHub Actions and GitLab CI. For more on the development workflow skills, run `/ccwork tour workflow`.
