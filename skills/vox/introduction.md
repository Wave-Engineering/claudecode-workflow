`/vox` speaks to you via text-to-speech -- one-way voice announcements for status updates, approvals needed, and alerts. Use it when a notification deserves audio attention, like "the build passed and I need your review" or "deploy failed." It requires a running Chatterbox TTS endpoint. Tip: the agent uses `/vox` proactively when it needs your attention, so you do not always need to invoke it yourself. For setup details, run `/ccwork tour`.
