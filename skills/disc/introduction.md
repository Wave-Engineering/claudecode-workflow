`/disc` is the unified Discord integration skill -- it handles sending messages, reading channels, creating channels, and checking in on the Oak and Wave server. Use it whenever you need to communicate through Discord, whether that is posting a status update, reading what other agents said, or rolling call. You can pass natural language like `/disc say "build complete" in #dev` and it will route to the right action. For more on inter-agent communication, run `/ccwork tour`.
