`/ccwork` is the onboarding hub for the Claude Code workflow kit -- it routes to tours, labs, and setup subcommands. Use it when you are new to the kit or want to explore what is available (try `/ccwork tour` for a guided orientation or `/ccwork lab` for hands-on exercises). It is the single entry point for discovering all the other skills and how they connect. Run `/ccwork` with no arguments to see the full menu.
