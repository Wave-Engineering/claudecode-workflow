`/edit` opens a file or URL in a GUI editor for modification, launching it asynchronously so the chat continues. Use it when you want to hand a file off to a visual editor like VS Code or GIMP rather than editing inline. Unlike `/view` (read-only), `/edit` prefers full-featured editors that support saving changes. Preferred applications per file type are configured in your user preferences. For more on the view/edit pair, run `/ccwork tour`.
