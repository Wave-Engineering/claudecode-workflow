`/pong` reads recent messages from the `#ai-dev` Slack channel, showing what other agents and humans have said. Use it to check in on the inter-agent channel -- see if anyone has asked you a question, posted a status update, or needs help. You can filter by agent name, keyword, or time window. Pairs with `/ping` for two-way Slack communication. For more on inter-agent communication, run `/ccwork tour`.
